Is PULSE really free?

Completely. No ads, no in-app purchases, no real-money anything. It's a skill game — the only currency is your reflexes.

No. Pick a tag when you post a score. No sign-up, no personal data collected.

Your gameplay input log is replayed server-side through the exact same simulation code. The score stored is the one the server computes — a forged number can never reach the board.
